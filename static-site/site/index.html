Zero Runtime Code

Pure static assets with long‑lived caching and a strict Content‑Security‑Policy.

Non‑root by Design

Unprivileged Nginx on port 8080, read‑only FS, tmpfs for runtime dirs, caps dropped.

Multi‑Arch Images

Buildx targets amd64, arm64, and armv7 for Odroid/RPi clusters.

Defense‑in‑Depth

  • Content‑Security‑Policy + strict headers
  • Read‑only container FS
  • Drop ALL Linux capabilities
  • No shell or extra packages in final image

Operational Safety

  • Healthchecks and start‑first updates
  • Immutable cache for assets
  • Minimal surface area

Deploy

  1. Build & push a multi‑arch image with Make release.
  2. docker stack deploy -c stack.yaml web
  3. Place behind your reverse proxy/TLS if desired.
